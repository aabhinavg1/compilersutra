GPU compilers are where compiler theory meets hardware constraints in a very visible way. This track connects device architecture, memory behavior, and kernel execution to practical programming and optimization choices.

## Who This Is For

- engineers exploring GPU compiler stacks
- developers learning OpenCL and heterogeneous programming
- readers who want to understand why parallel hardware changes compiler decisions

## Outcome

You should be able to explain the programming model, set up OpenCL workflows, write basic kernels, and reason about why compiler and memory decisions matter on GPUs.
