## Recommended Use

- Start with `Compilers` if frontend, IR, SSA, and backend flow are still fuzzy.
- Move into `LLVM` after compiler basics if you already think in passes and `.ll` files.
- Use `MLIR` when you want to understand staged lowering and dialect design.
- Use `AI in Compilers` when you care about learned heuristics and AI-assisted compiler decisions.
- Use `Compilers in AI` when you want the model-to-kernel execution path.
