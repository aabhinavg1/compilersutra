# Instruction Flow in a Modern CPU MCQs Quiz

<Question question="1). A useful high-level model of instruction flow in a modern CPU is:" options={['Fetch -> Sleep -> Compile -> Link', 'Fetch -> Decode -> Rename -> Dispatch/Schedule -> Execute -> Writeback -> Retire', 'Parse -> Tokenize -> Optimize -> Ship', 'Load -> Save -> Print -> Exit']} answer="Fetch -> Decode -> Rename -> Dispatch/Schedule -> Execute -> Writeback -> Retire" />
<Question question="2). Which part of the CPU is most associated with fetch and decode?" options={['Frontend', 'Backend only', 'Linker', 'Heap allocator']} answer="Frontend" />
<Question question="3). Which part of the CPU is most associated with scheduling and execution?" options={['Frontend', 'Backend', 'Preprocessor', 'Source parser']} answer="Backend" />
<Question question="4). Why is register renaming useful?" options={['It removes all true data dependencies', 'It helps remove false name-based dependencies', 'It changes the ISA at runtime', 'It disables writeback']} answer="It helps remove false name-based dependencies" />
<Question question="5). What kind of dependency can rename NOT remove?" options={['True data dependency', 'False naming dependency', 'Write-after-write name reuse', 'Write-after-read name reuse']} answer="True data dependency" />
<Question question="6). The scheduler mainly tries to answer:" options={['Which source file should compile next?', 'Which instructions are ready and which resource can run them now?', 'Which comments should be removed?', 'Which register is largest in memory?']} answer="Which instructions are ready and which resource can run them now?" />
<Question question="7). Which statement is correct?" options={['Instructions must always execute in source order', 'Instructions may execute out of order but usually retire in order', 'Retirement always happens before execution', 'Decode only happens after writeback']} answer="Instructions may execute out of order but usually retire in order" />
<Question question="8). Writeback is important mainly because:" options={['It turns source code into assembly', 'It often makes dependent instructions become ready', 'It removes the need for decode', 'It enlarges the instruction cache']} answer="It often makes dependent instructions become ready" />
<Question question="9). Retirement is the stage where:" options={['The CPU commits results architecturally in safe program order', 'The binary is linked again', 'All branch predictors are disabled', 'The compiler inserts comments']} answer="The CPU commits results architecturally in safe program order" />

What is the key difference between execute and retire?" options={['They are the same thing', 'Execute is internal work; retire is architectural commitment', 'Retire happens before fetch', 'Execute belongs to software and retire belongs to the linker']} answer="Execute is internal work; retire is architectural commitment" />
