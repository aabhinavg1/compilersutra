# Architecture vs Organization MCQs Quiz

<Question question="1). Computer architecture is best described as:" options={['The internal transistor layout only', 'The software-visible contract of the machine', 'The cooling system of the CPU package', 'The benchmark score of the processor']} answer="The software-visible contract of the machine" />
<Question question="2). Computer organization is best described as:" options={['The way hardware is arranged internally to implement the architecture', 'The source language grammar', 'The linker command line', 'The operating system shell']} answer="The way hardware is arranged internally to implement the architecture" />
<Question question="3). Which of the following belongs more directly to architecture?" options={['Branch predictor design', 'Execution port layout', 'ISA-visible registers and memory-ordering guarantees', 'Reorder buffer size']} answer="ISA-visible registers and memory-ordering guarantees" />
<Question question="4). Which of the following belongs more directly to organization?" options={['Instruction set semantics', 'Calling convention contract', 'Pipeline, cache hierarchy, and issue queues', 'The legal meaning of ADD in the ISA']} answer="Pipeline, cache hierarchy, and issue queues" />
<Question question="5). Why is the memory model considered part of architecture?" options={['Because software must know the ordering and visibility guarantees it can rely on', 'Because it only affects transistor placement', 'Because it is chosen by the linker', 'Because caches do not exist']} answer="Because software must know the ordering and visibility guarantees it can rely on" />
<Question question="6). <Question question="7). If two CPUs both implement `x86-64`, what can still differ a lot?" options={['The visible meaning of ADD in the ISA', 'The internal microarchitecture and resulting performance', 'Whether machine code exists', 'Whether registers are part of architecture']} answer="The internal microarchitecture and resulting performance" />
<Question question="9). Skylake, Zen 2, and Cortex-X1 are examples of:" options={['Programming languages', 'Microarchitectures or organizational designs', 'Linker scripts', 'Only memory allocators']} answer="Microarchitectures or organizational designs" />
<Question question="10). If a simple integer ADD has similar nominal latency on multiple CPUs, program performance can still differ because:" options={['The surrounding frontend, scheduler, cache, and branch machinery can differ', 'Latency is the only thing that ever matters', 'The ISA stops working on some CPUs', 'Registers disappear during execution']} answer="The surrounding frontend, scheduler, cache, and branch machinery can differ" />
