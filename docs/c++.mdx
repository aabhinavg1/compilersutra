## What You Will Learn

This roadmap is organized around the actual progression a serious C++ learner needs:

1. Computer and compilation basics
2. Core language fundamentals
3. Functions, pointers, references, and memory
4. Object-oriented programming
5. Templates and STL
6. Resource management and modern C++
7. Concurrency and performance-aware code
8. Systems programming and tooling
9. Modern language features such as concepts and ranges
10. Interview prep and professional development

## How To Use This Hub

- If you are new to C++, start with the Level 0 and Level 1 pages before touching templates or concurrency.
- If you already know syntax but feel weak in code design, start at Level 3 and Level 4.
- If you work in systems, compilers, or performance engineering, prioritize Levels 6 through 9.
- If your goal is interviews, do not skip memory, STL, and concurrency just because they feel harder.
